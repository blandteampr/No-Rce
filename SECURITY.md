Security Policy
Supported Versions

The following versions of Bland™ are currently supported with security updates:

Version	Supported
5.1.x	✅
5.0.x	❌
4.0.x	✅
< 4.0	❌

We strongly recommend using the latest supported version to ensure maximum security.

Reporting a Vulnerability

We take the security of Bland™ seriously and appreciate responsible disclosure.

How to Report

Please report security vulnerabilities using GitHub Security Advisories (preferred).

Alternatively, you may open a private issue on GitHub for sensitive reports.

What to Expect

Acknowledgement: Within 48 hours of receiving the report.

Assessment: The vulnerability will be reviewed and validated.

Outcome:

If accepted, a fix will be developed and released in a security update.

If declined, we will provide a clear explanation.

⚠️ Please do not publicly disclose any vulnerability details until an official fix has been released, to protect users and the project.
